# Train Ticket Booking System

A multi-role web application for managing and booking train tickets, 
built as an academic project.

## Features
- User panel: browse available trains, routes, and travel schedules
- User panel: book tickets based on available seats
- Admin panel: manage cities, routes, trains, destinations, and schedules
- Admin panel: manage travel hours and available seat capacity
- Booking and reservation management

## Tech Stack
- PHP (raw, no framework)
- MySQL
- HTML / CSS / Bootstrap

## Roles
- Admin: full control over trains, routes, cities, and schedules
- User: browse available trips and book tickets

## Setup
1. Clone the repo
2. Import the SQL file from the `/database` folder into MySQL (via phpMyAdmin)
3. Copy `conn.example.php` to `conn.php` and set your database credentials
4. Run via XAMPP or any local PHP server
